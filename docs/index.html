A tiny vault for project fleets.

Keep secrets secure and environments in sync without cloud dependencies.

Encrypted vault

Each project manages its own secrets.enc.json. Built on trusted industry standards, secrets stay encrypted at rest via SOPS and Age.

Global secret reuse

Avoid duplicating critical variables. Define shared global tokens in a central namespace and safely reference them across your configurations.

Template injection

Feed placeholders like {{GLOBAL.KEY}} into your local .env.template files. Generate fresh environment files in a single click.

GitOps from the UI

Native integration right within the system. Check repository status, review encrypted changes, and stage, commit, or push directly to git.

Run the console locally.

A few simple steps to get you up and running with Rage UI.

setup.sh 
git clone https://github.com/Sofian-bll/Rage-UI.git
cd Rage-UI/backend && bun install && bun run server.ts
cd Rage-UI/frontend && npm install && npm run dev
env-merger-output 
# .env.template
DB_URL=postgresql://{{GLOBAL.DB_USER}}:{{DB_PASS}}@localhost:5432
API_KEY={{GLOBAL.API_TOKEN}}
