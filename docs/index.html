A tiny vault for project fleets.

Rage UI is deliberately scoped for one trusted operator managing a set of local or homelab projects.

01 / vault

Encrypted project stores

Each project keeps a `secrets.enc.json` file encrypted by SOPS/Age, so Git stores ciphertext instead of raw values.

02 / global

Shared secret reuse

Put common tokens in `global/`, reference them from many templates, and rotate once instead of editing every project.

03 / inject

Template-driven `.env` files

Use `{{GLOBAL.KEY}}` and `{{KEY}}` placeholders to generate real `.env` files from encrypted inputs.

04 / sync

GitOps workflow

The UI exposes repository status and sync actions so encrypted changes can move through a normal Git remote.

Run the console locally.

Start the Bun backend and Vite frontend in separate terminals, then open the web UI.

git clone https://github.com/Sofian-bll/Rage-UI.git
cd Rage-UI/backend
bun install
bun run server.ts
# second terminal
cd Rage-UI/frontend
npm install
npm run dev
