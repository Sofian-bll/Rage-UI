Encrypted vault

Each project maintains its own secrets.enc.json. SOPS/Age ensures data is only readable by those with the master key.

Global secret reuse

Centralize shared tokens in global/. Reference them from any project to eliminate duplication and update errors.

Template injection

Define .env.template with placeholders. One click merges global and local values into a ready-to-use .env file.

GitOps from the UI

Check repository status, view diffs, commit changes, and push updates directly from the dashboard. Seamless version control.

Setup 
git clone https://github.com/Sofian-bll/Rage-UI.git
cd Rage-UI/backend && bun install && bun run server.ts
cd ../frontend && npm install && npm run dev
Injection Logic
# .env.template
DB_URL=postgres://{{GLOBAL.USER}}@{{HOST}}
# Generated .env
DB_URL=postgres://admin@localhost
